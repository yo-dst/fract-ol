Fractals availables :<br/>
	- Mandelbrot<br/>
	- Julia<br/>
	- Burning Ship<br/>
<br/>
Commands :<br/>
	- P 			=> Switch colors<br/>
	- Left click 	=> Increase iterations<br/>
	- Right click 	=> Decrease iterations<br/>
	- Scroll 		=> Zoom<br/>
	- V 			=> Turn on/off Julia point selection<br/>
	- R 			=> Reset fractal<br/>
	- ESC			=> Exit<br/>
